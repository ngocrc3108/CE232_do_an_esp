# Những việc đã làm được:
- Kết nối wifi.
- Kết nối được mqtt.
- Hàm bật tắt led (ledSetState).
- Xử lí được yêu cầu bật/tắt led từ server (ledEventHandler).
# Những việc cần làm:
- Hàm đóng/mở cửa. (doorSetState).
- Hàm bật/tắt quạt. (fanSetState).
- Hàm chỉnh tốc độ quạt. (fanSetLevel).
